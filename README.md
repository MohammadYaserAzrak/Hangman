# Hangman
The well-known game, Hangman, built using turtle. It uses the SOWPODS word list.

## Prerequisites
- Python 3.x
  
## How to Run

1. Clone this repository:
```
git clone <repo-link>
```
2. Run the main.py file:
```
python main.py
```
